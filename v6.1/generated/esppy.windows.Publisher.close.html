esppy.windows.Publisher.close¶

Publisher.close()¶

Close the web socket connection

Examples

Create the publisher instance using CSV and an event rate of 200 events per millisecond.

>>> pub = Publisher(window, rate=200)

Send the CSV data.

>>> pub.send('1,2,3')

Close the connection.

>>> pub.close()
