Installation
============

The ESPPy package can be installed in multiple ways as described below.

Pip
---

You can install ESPPy using the ``pip`` command available in most Python distributions::

    pip install sas-esppy


Conda
-----

If you are using the Anaconda distribution of Python, you can use the ``conda`` command
to install ESPPy::

    conda install -c sas-institute sas-esppy


GitHub
------

ESPPy can be installed using ``pip`` and the release files at
`<https://github.com/sassoftware/python-esppy/releases>`_.
